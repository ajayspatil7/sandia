# Sandia - AI-Powered Cybersecurity Analysis Platform

A modern, world-class web application for cybersecurity file analysis using AI agents and cloud-based processing.

## Features

- **Premium UI/UX**: Designed with 40+ years of experience from top tech companies
- **AI-Powered Analysis**: Static and dynamic analysis using state-of-the-art LLMs
- **Real-time Processing**: Live progress tracking and results visualization
- **Secure Cloud Environment**: Isolated VM processing with golden snapshot restoration
- **Professional Design**: Modern cybersecurity-themed interface with animations

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Styling**: Custom cybersecurity theme with gradient effects

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint

# Format code
npm run format
```
